# nanolog

> "pensado para guardar o pouco que ainda resta de mim"

segundo cérebro pessoal estruturado em markdown, sincronizado com obsidian.
não é um sistema perfeito — é um sistema honesto.

---

## o que é isso

um lugar pra registrar o que penso, o que aprendo, o que sinto e o que quero fazer.
dividido entre o lado humano — diário, devaneios, memórias, sonhos — 
e o lado estratégico — conhecimento, habilidades, projetos, modelos mentais.

---

## estrutura

| pasta | propósito |
|---|---|
| 00. inbox | captura rápida, sem julgamento |
| 01. diário | registro do dia |
| 02. devaneios | pensamentos livres |
| 03. ideias | pensamentos com intenção |
| 04. memórias | o que vale preservar |
| 05. sonhos | o que acontece quando durmo |
| 06. notas de amor | o que sinto por quem importa |
| 07. biblioteca | referências externas |
| 08. arquivo | o que saiu do fluxo ativo |
| 09. mente | conhecimento e ação |

taxonomia completa em [internal.md](./internal.md)
